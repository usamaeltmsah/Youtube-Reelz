# Youtube-Reelz

This is a simple iOS app that display a YouTube playlist in a collection view, with clicking on any item will play the YouTube video.

## Features

- [x] Usage of MVVM design pattern
- [x] Usage of Moya
- [x] Usage of PromiseKit
- [x] Usage of RSwift

## Requirements

- Swift 5+
- Xcode 12+
